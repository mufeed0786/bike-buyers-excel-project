
# Bike Buyers Excel Dashboard

## Project Overview
Analyzed customer demographics and buying patterns using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Dashboard

## Key Insights
- Analyzed bike purchase trends by gender.
- Studied the relationship between income and bike purchases.
- Examined age groups and commute distance patterns.

## Features
- Data Cleaning
- Pivot Tables
- Interactive Dashboard
- Data Visualization

## Dataset
Bike Buyers Dataset
